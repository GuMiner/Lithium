If you'll want to download these models for yourself, be prepared for some complications!

  • First, you'll need to use the Internet Archive Wayback Machine to acquire the files.
  • After that, BRL-CAD is one of the few programs that can properly convert the unusual 3D model structure into .STL files for import into Autodesk 123D Make.
